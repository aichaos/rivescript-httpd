RiveScript::HTTPd

Welcome!

If you're seeing this page, it means that RiveScript::HTTPd seems to be working! RiveScript::HTTPd is a simple web server for quickly getting started with RiveScript chatbot development!

This server is only for your development and testing purposes. It is not meant to be publically accessible to the world. By default the server only listens on your local loopback address (127.0.0.1). When you've got your chatbot configured how you like it, you can upload it and one of the CGI scripts to a web host and let the world see it. If your web host doesn't allow you to run neither Perl nor Python CGI scripts, you can upload the JavaScript version instead! Learn more

Edit Your Bot

To edit your bot's replies, open the public_html folder in your RiveScript::HTTPd directory and then open the brain folder. The replies are stored within *.rs files.

By default, RiveScript::HTTPd comes with the standard demo brain that includes a lot of Eliza-like responses. You can open these files in a text editor such as Notepad to begin editing replies. After saving the changes, reload the chat page in your web browser to reload the bot. If you add new files, you'll need to refresh the file list for the JavaScript version to get updated.
