What can I do?

If this URL ends with ".py", it's probably a Python script! You can download Python from Python.org and install it on your computer. If you're on Windows, it will probably install to a path like C:\Python27\python.exe, and Python scripts should start working on this web server after it's installed.

If the URL ends with ".pl", it's probably a Perl script! You can download Perl from ActivePerl.com and install it on your computer. If you're on Windows, it will probably install to a path like C:\Perl, and Perl scripts should start working on this web server after it's installed.
